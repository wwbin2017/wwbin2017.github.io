python 命令行参数解析

getopt分长格式和短格式，代码中带冒号“：”和等号“=”的表示后面有参数

返回值 options 是个包含元组的列表，每个元组是分析出来的格式信息，比如 [(‘-i’,’localhost’),(‘-p’,’8000’)] ;
args 是个列表，包含那些没有‘-’或‘–’的参数，比如：[‘100’,’200’]

注意：定义命令行参数时，要先定义带’-‘选项的参数，然后定义没有‘-’的参数
