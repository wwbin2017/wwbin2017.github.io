Jump Game II

发表于 2018-12-09

LeetCode 题目：Jump Game II 难度：hard

我的DP解法，竟然超时了。还是讨论区厉害，居然贪心算法。也是没有仔细分析问题。

根据讨论区写的解法，其实写出来就懂了，感觉就是没有好好思考问题。解释如下：因为可以jump到（1，num[i]）。因此，从第一个开始跳，能跳的最大范围是（1，num[0]）。第二跳的区域肯定在（1，num[0]），因此选择这个区域最大的跳数，curFarthest。第三跳的范围可定在（num[0]，curFarthest），如此下去。
