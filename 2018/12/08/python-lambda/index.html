python lambda表达式

lambda表达式，通常是在需要一个函数，但是又不想单独再定义个函数，也就是匿名函数，匿名函数应该尽量简单
Python提供了很多函数式编程的特性，如：map、reduce、filter、sorted等这些函数都支持函数作为参数，lambda函数就可以应用在函数式编程中。
